# 多框分割功能使用说明

## 功能概述

本应用支持三种分割场景：

1. **仅 WG 提示框**：只分割 WG（全腺体）区域，返回红色 mask
2. **仅 CG 提示框**：只分割 CG（中央腺）区域，返回绿色 mask
3. **同时有 WG 和 CG 提示框**：分割 WG、CG、PZ（外周区）三个区域，分别返回红色、绿色、蓝色 mask

## 快速开始

### 方法 1：使用示例图像（推荐新手）

1. **加载示例图像**
   - 点击左侧面板的 "Load Sample Image" 按钮
   - 系统会自动加载一张预设的医学图像（1176 x 993 像素）

2. **选择框类型**
   - 在 "Box Type Selection" 中选择框类型：
     - **WG**（蓝色）：用于标记全腺体区域
     - **CG**（橙色）：用于标记中央腺区域

3. **绘制框**
   - 在图像上点击并拖动绘制框
   - 可以绘制多个不同类型的框
   - 框的颜色会根据类型变化：
     - WG 框：蓝色边框 + 蓝色标签
     - CG 框：橙色边框 + 橙色标签

4. **管理框**
   - 左侧面板会显示所有已绘制的框列表
   - 点击垃圾桶图标可以删除单个框
   - 点击 "Clear All" 清除所有框

5. **运行分割**
   - 点击 "Run Segmentation" 按钮
   - 等待处理完成（通常几秒钟）

6. **查看结果**
   - 分割结果会以不同颜色的 mask 叠加显示在原图上：
     - WG mask：红色半透明
     - CG mask：绿色半透明
     - PZ mask：蓝色半透明
   - 下方会显示成功信息和各个区域的说明

### 方法 2：上传自己的图像

1. 点击 "Upload Image" 按钮
2. 选择本地医学图像文件（支持 JPG、PNG 等格式）
3. 按照上述步骤 2-6 进行操作

## 分割场景说明

### 场景 1：仅分割 WG（全腺体）

**适用情况**：只需要分割整个前列腺腺体区域

**操作步骤**：
1. 选择 "WG" 框类型
2. 在图像上绘制一个包含整个腺体的框
3. 点击 "Run Segmentation"

**结果**：
- 返回 WG mask（红色）

### 场景 2：仅分割 CG（中央腺）

**适用情况**：只需要分割中央腺区域

**操作步骤**：
1. 选择 "CG" 框类型
2. 在图像上绘制一个包含中央腺的框
3. 点击 "Run Segmentation"

**结果**：
- 返回 CG mask（绿色）

### 场景 3：同时分割 WG、CG、PZ（三区域）

**适用情况**：需要同时分析全腺体、中央腺和外周区

**操作步骤**：
1. 先选择 "WG" 框类型，绘制一个包含整个腺体的框
2. 再选择 "CG" 框类型，在腺体内部绘制一个包含中央腺的框
3. 点击 "Run Segmentation"

**结果**：
- 返回三个 mask：
  - WG mask（红色）：全腺体
  - CG mask（绿色）：中央腺
  - PZ mask（蓝色）：外周区（通过 WG - CG 计算）

**注意**：PZ（外周区）是通过数学计算得出的：PZ = WG - CG

## 模式选择

### Basic Mode
- 使用标准 SAM2 模型
- 适用于通用图像分割

### Medical Mode（推荐）
- 使用医学优化的 SAM2 模型
- 针对医学图像进行了优化
- 分割效果更好

## 提示与技巧

1. **框的绘制技巧**：
   - 框应该稍微大一点，包含目标区域
   - 避免框太小，可能导致分割不完整
   - 对于三区域分割，CG 框应该在 WG 框内部

2. **多框管理**：
   - 可以绘制多个相同类型的框
   - 系统会使用第一个框作为提示
   - 不需要的框可以随时删除

3. **结果解读**：
   - 红色区域：全腺体（WG）
   - 绿色区域：中央腺（CG）
   - 蓝色区域：外周区（PZ）
   - Mask 是半透明的，可以叠加查看原始图像

4. **错误处理**：
   - 如果绘制框时出错，点击 "Clear All" 重新开始
   - 如果分割结果不理想，尝试调整框的位置和大小
   - 确保选择了正确的框类型（WG 或 CG）

## 技术说明

### 颜色规范
- WG mask：RGB (255, 0, 0)，Alpha 150
- CG mask：RGB (0, 255, 0)，Alpha 150
- PZ mask：RGB (0, 0, 255)，Alpha 150

### 框颜色规范
- WG 框：蓝色边框 (#3b82f6)，蓝色背景
- CG 框：橙色边框 (#f97316)，橙色背景

### 数据格式
- 输入：Base64 编码的图像 + 框坐标数组
- 输出：多个 Base64 编码的 PNG mask 图像

## 常见问题

**Q: 为什么我的分割结果显示为方形？**
A: 当前使用的是 Mock 模式（演示模式），返回的是示例性的方形 mask。要获得真实的医学分割结果，需要配置真实的 Python 模型。

**Q: 如何启用真实的医学分割？**
A: 需要在 `.env.local` 文件中设置 `USE_PYTHON_SEGMENTATION=true`，并确保模型文件已准备好。详见 `SUMMARY.md` 文档。

**Q: PZ 是如何计算的？**
A: PZ（外周区）是通过数学计算得出的：PZ = WG - CG。即外周区 = 全腺体 - 中央腺。

**Q: 可以绘制多个相同类型的框吗？**
A: 可以，但系统目前只使用第一个框作为分割提示。

**Q: 如何删除某个框？**
A: 在左侧面板的 "Boxes" 列表中，点击对应框右侧的垃圾桶图标。
